タイトル

RopeAction

-----------------------------------------------------------------------------------------------------------------------------------------------------

概要

3年生のときに学校のNintendoSwitch開発で制作したゲームです。

-----------------------------------------------------------------------------------------------------------------------------------------------------

開発環境と制作ツール

開発環境	… Unity 2022.3.28f1

-----------------------------------------------------------------------------------------------------------------------------------------------------

担当した箇所

ソースコード … Assetsファイル内のYoshinoファイル内のすべて

使用したアセット

FinalIK

ObiRope

-----------------------------------------------------------------------------------------------------------------------------------------------------

制作開始日付
2024年 4/17

制作期間
4/17～現在制作中

-----------------------------------------------------------------------------------------------------------------------------------------------------

操作方法

キーボード操作

Wキー  　 … 前進
Sキー     … 後退
Aキー　　 … 左移動
Dキー　　 … 右移動
Spaceキー … ジャンプ
マウス　　… カメラ操作
右クリック… 画面上にカーソルが表示された場所へロープ発射
Vキー　　 … 近くのロープを掴む

ロープ発射中
右クリック… ロープを解除
マウスホイール　… ロープの長さを変更
Spaceキー … ジャンプ
Vキー　　 … 掴んでいるロープを離す
Eキー     … カーソルが表示された場所とロープを繋ぐ

-----------------------------------------------------------------------------------------------------------------------------------------------------

ゲーム説明
ロープを駆使してステージを攻略していくアクションゲームです。
現在開発中のためステージ数が少ない状態です。

-----------------------------------------------------------------------------------------------------------------------------------------------------

コンセプト
ObiRopeと物理演算の相性がいいので、ObiRope単体でどこまで面白さが出せるか試したい

-----------------------------------------------------------------------------------------------------------------------------------------------------

アピールポイント
ObiRopeとFinalIKをを組み合わせて、説得力のあるプレイヤーの挙動を実現しています。
ObiRopeのサンプルのプログラムを改変して、変更しやすいかたちでロープのプログラムを実装しています。
